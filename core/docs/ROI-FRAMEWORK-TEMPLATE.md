# ROI Framework Template — Splashworks Projects

> Use this template at project kickoff and completion to quantify value delivered.
> Every Splashworks project should produce a portfolio-ready ROI metric card.

---

## Instructions

1. **At kickoff:** Fill in Section A (Baseline) — estimate the manual effort this project replaces
2. **During execution:** Track Section B (Investment) — log actual hours and costs per session
3. **At completion:** Fill in Section C (Results) and Section D (Reuse) — calculate the ROI
4. **For portfolio:** Copy Section E (Summary Card) into the project README or case study
