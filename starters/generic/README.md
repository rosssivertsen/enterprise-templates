# project-name

Generic project starter from enterprise-templates.

## Getting Started

This is a minimal project skeleton. Customize the `Makefile` targets for your language and toolchain.

## Available Commands

Run `make help` to see all available commands.

## Customization

1. Update the `Makefile` with your language-specific build, test, and run commands
2. Add your source code to `src/`
3. Add your tests to `tests/`
4. If using Docker, copy the appropriate Dockerfile from enterprise-templates
